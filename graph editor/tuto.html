But du projet

Le but du graph editor est de structurer la narration et l'enchaînnement des scènes sous forme de graphe.

Le joueur commence sur la carte de Paris.

Les destinations à visiter sont déterminées par le parcours du graphe.

Nouvelle scène

Pour créer une scène, il faut utiliser la fenêtre flotante repositionnable

label

label est le nom de la scène. Il doit impérativement commencer par "scene_"

vignette

La vignette est l'image qui sera visible sur la carte de Paris

Coordonnées

Les coordonnées sont les coordonnées d'affichage de la vignette sur la carte de Paris

Utilisez coordinates.html pour les calculer

Label Regex

Label regex est une expression régulière qui permet d'évaluer si la scène est disponible sur la carte de Paris.

Par exemple, la regex "root" indique que la scène est disponible dès le début de l'aventure.

Une regex peut cominer plusieurs scènes. Si la scène doit s'afficher si le joueur a complété la scène A ou la scène B. Une regex peut résumer cela par la regex: "scene_a | scene_b"

Enfin, le bouton "Créer nouvelle" permet de créer la scène.

Une fois crée, celle-ci peut être déplacer pour plus de lisibilité du graphe.

Les notes

Les notes d'une scène ne sont pas utilisées dans le jeu.

Elles permettent aux scénaristes d'ajouter un commentaire ou un résumé de la scène afin de mieux comprendre ce que chaque scène fait.

Pour des raisons de simplicité, il est interdit d'utiliser des guillemets dans les notes !

Modifier une scène

Il est possible de modifier une scène en cliquant sur le nom de celle-ci.

Une fois les propriétés modifiées, le bouton "Mettre à jour" permet de modifier la scène.

Le label ne peut pas être modifiés pour l'instant. Pour cela, cliquez sur le nom de la scène pour l'éditer, changez le nom et créez une nouvelle scène, puis supprimez l'ancienne.

Supprimer une scène

Pour supprimer une scène, cliquez sur le nom de la scène et cliquez sur le bouton 'Supprimer"

Nouveau Bool

Pour Créer un Bool, renseigner les champs et cliquez sur "Nouveau"

Label parent

Le nom du label de la scène dans laquelle le booléen est affecté (choix crucial qui changera la suite)

Nom du Bool

Un nom parlant pour le booléen

Modifier Bool

Pour modifier un Bool, cliquez sur le nom du Bool et cliquez sur le bouton 'Supprimer"

Supprimer Bool

Pour supprimer un Bool, cliquez sur le nom du Bool et cliquez sur le bouton 'Supprimer"

Sauvegarder/Charger un graphe

Il y a aura plusieurs graphe dans le jeu. Le premier étant la trame de base du joueur et d'autres lorsque surviennent des BDAs.

Pour charger un graphe existant, il suffit de cliquer sur "Charger" et de sélectionner le fichier à lire.

Pour des raisons de sécurité, un navigateur web ne peut pas modifier les fichiers sur l'ordinateur.

Ainsi, pour sauvegarder notre travail, le bouton "Sauvegarder" crée et télécharge un fichier avec le même nom que celui qui a été chargé.

Il vous revient de déplacer le fichier depuis votre dossier téléchargement dans le dossier du projet.

Recherche dans le graphe

Lorsque le projet nécessitera un graphe volumineux, il sera possible de faire une recherche de scène à l'aide du bouton "Chercher" en haut de la fenêtre flotante.

Vérifier les erreurs

Il est possible d'effectuer une recherche d'erreurs dans le graphe afin de faciliter la maintenance du graphe
