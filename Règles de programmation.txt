Labels:
Concernant les label, ren'py recherche dans tous les fichiers.
Ce qui est pratique mais pour ne pas s'y perdre lorsque le projet sera volumineux,
nommer la variable par <nom de fichier>_<nom du label>.
Par exemple: map_display (fichier map, label map_display). 


Scenes:
Pareil pour les scene.
Nommer le fichier rpy avec le prefixe <scene>_<nom de la scene>
Par exemple: scene_map


Le python n'accepte pas les tabulations, donc pensez à indenter avec des espaces.
